# Audit Trail

All significant operations are logged here.

## Logging
- Backup operations → change.log
- Deploy operations → change.log
- Recovery operations → change.log
- Identity access → this file
- Safe mode events → this file

## Entry Format
| Timestamp | Action | User/Source | Details | Result |

## Retention
- Keep for 90 days minimum
- Archive after 90 days
- Never delete before archive
