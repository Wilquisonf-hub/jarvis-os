# Projects Template

> Active projects are managed in PROJECTS.md (root workspace file).

---

## Project Fields

| Field | Description |
|------|-|
| Name | Project name |
| Owner | Project lead |
| Status | ACTIVE, PAUSED, COMPLETED |
| Priority | URGENT, HIGH, MEDIUM, LOW |
| Deadline | Project deadline |
| Key Contacts | Who to coordinate with |
| Next Steps | What's next |

*Keep PROJECTS.md in workspace root for simplicity*
