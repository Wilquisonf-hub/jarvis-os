## Task Fields

| Field | Description |
|------|---|
| # | Task ID |
| Task | Name and description |
| Owner | Person responsible |
| Status | PENDING, IN_PROGRESS, WAITING, COMPLETED, DELAYED, BLOCKED, SCHEDULED |
| Priority | URGENT, HIGH, MEDIUM, LOW, SCHEDULED |
| Deadline | Due date |
| Notes | Context and updates |
