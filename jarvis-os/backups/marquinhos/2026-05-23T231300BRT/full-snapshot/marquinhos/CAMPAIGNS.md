# CAMPAIGNS.md - Marketing Campaign Registry

## Campaign Structure

- **ID** — unique campaign identifier
- **Name** — human-readable name
- **Type** — SEO, content, competitor, promotional, AI-search, etc.
- **Start Date**
- **End Date**
- **Status** — planning, active, paused, completed, cancelled
- **Objective** — what this campaign is trying to achieve
- **Target Metrics** — specific, measurable goals
- **Workers Used** — which workers are involved
- **Budget** — if applicable
- **Related Tasks** — link to TASKS.md entries
- **Notes**

## Campaign Types Reference

- **SEO** — technical SEO, site-wide improvements

- **AI-Search** — AI answer optimization, structured data

- **Link-Build** — internal and external linking campaigns
